# Open Source Contributions

A modern portfolio showcasing my open source contributions with interactive 3D carousels and electric spark effects.

## Features

- **Electric Spark Heading** - Animated voltage effects on hover
- **3D Rotating Carousels** - Interactive PR card displays for each program
- **Space Theme** - Floating astronaut and falling stars animation
- **Fully Responsive** - Optimized for mobile, tablet, and desktop
